# resume-analyzer

Resume Analyzer is a cloud-native AI resume analysis platform built with Angular, Spring Boot, PostgreSQL, AWS S3, and LLM APIs. Users upload resumes, the backend stores files in S3, extracts resume text using PDF/Tika parsers, evaluates content through AI scoring workflows, and persists metadata, scores, section feedback, and user data in PostgreSQL. The application is secured with Spring Security and JWT, supports email verification, and is deployed to AWS EC2 using GitHub Actions and Docker Compose.
